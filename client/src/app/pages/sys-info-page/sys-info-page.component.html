О системе

Об интернет магазине

Интернет-магазин представляет собой цифровую платформу, на которой пользователи могут удобно просматривать ассортимент товаров, делать заказы и оплачивать покупки, не выходя из дома. Он предлагает множество функций, такие как фильтры для поиска, сортировка по различным критериям.

Инструкция по запуску проекта

  1. Шаг 1: Установите Node.js с официального сайта.
    • Перейдите по ссылке https://nodejs.org/
    • После установки проверьте версию Node.js командой node -v
  2. Шаг 2: Установите Angular CLI с помощью команды: npm install -g @angular/cli
  3. Шаг 3: Клонируйте проект и перейдите: git clone "ссылка на репозиторий"
  4. Шаг 4: Перейдите в его папку: cd путь/к/проекту"
  5. Шаг 5: Установите зависимости проекта: npm install
  6. Шаг 6: Запустите проект командой: ng serve
  7. Шаг 7: Перейдите в браузер по адресу: http://localhost:4200

Как войти в систему

  • На главной странице сайта в шапке в правом верхнем углу находится кнопка «Войти».
  • По нажатию на эту кнопку откроется модальное окно для авторизации.
  • В модальном окне необходимо ввести свой Email и Пароль в соответствующие поля.
  • После ввода данных нажмите кнопку «Войти».
  • Обратите внимание, что администратор может войти только под уникальной комбинацией Email и пароля.
  • Если данные введены некорректно или учетная запись не существует, будет отображено сообщение об ошибке.

Как зарегистрироваться в системе:

  • Если у вас еще нет учетной записи на сайте, вам нужно пройти процесс регистрации.
  • В модальном окне входа есть ссылка с надписью «Зарегистрироваться». Нажмите на нее, чтобы открыть форму регистрации.
  • В открывшемся окне регистрации введите свой Email и Пароль в соответствующие поля.
  • После ввода данных нажмите кнопку «Зарегистрироваться».
  • Убедитесь, что введенные данные корректны, и не забывайте, что пароль должен быть безопасным.

Обзор главной страницы сайта

Страница состоит из трех частей: шапка, панель навигации, левая боковая панель и основная часть, где находится меню комплектующих.

В шапке представлены следующие функции:

  • «Войти» – для перехода на страницу авторизации пользователя, в случае если пользователь уже авторизовался, там располагается кнопка «Профиль». При нажатии на данную кнопку всплывает модальное окно, на котором расположены 2 ссылки: «Заказы» и «Выйти». При нажатии на ссылку «Заказы» открывается страница со всеми заказами пользователя. При нажатии на ссылку «Выйти» система закрывает доступ.
  • «О системе» - для предоставления информации о системе.
  • «О разработчиках» - для предоставления информации о разработчиках.
  • "Корзина" - для просмотра добавленных товаров, их количества, итоговой стоимости и стоимости каждого товара.

В основной части представлены следующие элементы интерфейса:

  • «Поиск» – для отображения комплектующего, введённого в поле для поиска.
  • Панель навигации, на которой отображается список представленных комплектующих, при нажатии на название любого комплектующего открывается страница с данным типом комплектующих.
  • Каталог комплектующих, где в характеристику каждого входит его название, краткое описание и цена, а также кнопка «Добавить», с помощью которой можно добавить выбранное комплектующее в корзину, при нажатии же на изображение комплектующего пользователю откроется модальное окно со справочной информацией о данном комплектующем.

Если пользователь нажмет на выбранное комплектующее из списка, то откроется модальное окно комплектующего с подробными характеристиками. Здесь будет отображаться соответствующая информация: название комплектующего, описание, рейтинг, гарантия, производитель, изображение. После информации о товаре находится кнопка "Добавить в корзину". При нажатии на эту кнопку соответствующая позиция будет добавлена в корзину.

Пример изображения модального окна комплектующего:

В боковой панели представлены следующие элементы интерфейса:

  • Фильтрация по цене - пользователи могут выбрать комплектующие по цене или задать диапазон цен, в котором будет находится цена комплектующего.
  • Фильтрация по гарантии - пользователи могут выбрать комплектующие по гарантии или задать диапазон гарантии, в котором будет находится гарантия комплектующего.
  • Фильтрация по производителю - пользователи могут выбрать комплектующие по производителю.
  • Кнопка "Применить" - по нажатию на данную кнопку приминяются выбарнные фильтры и система перерисовывает каталог комплектующих в соответсвии с выбранными фильтрами.
  • Сортировка по цене - пользователи могут отсортировать каталог комплектующих по цене по возрастанию и в обратном порядке.
  • Сортировка по производителю - пользователи могут отсортировать каталог комплектующих по производителю в алфавитном порядке.
  • Сортировка по гарантии - пользователи могут отсортировать каталог комплектующих по гарантии по возрастанию и в обратном порядке.
  • Кнопка "Сбросить" - по нажатию на данную кнопку отменяются выбарнные фильтры и сортировка, а также система перерисовывает каталог комплектующих, возвращая первоначальный вариант.

Обзор страницы с корзиной

Основная часть данной страницы состоит из трёх разделов: корзина, персональная информация и адрес доставки.

На данной странице можно просмотреть добавленные товары, их количество, итоговую стоимость и стоимость каждого товара, а также в данном пункте будут представлены следующие функции:

  • "Очистить корзину" - для удаления всех товаров из заказа.
  • "+" и "-" - для изменения количества данного товара в заказе.
  • "х" - для удаления товара из заказа.
  • "Перейти к оплате" – для создания заказа, после нажатия данного элемента заказ переместиться на страницу текущие заказы.
